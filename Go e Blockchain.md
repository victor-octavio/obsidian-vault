SDK Go Hyperledger Fabric -> `https://github.com/hyperledger/fabric-sdk-go`

Existem algumas opções para trabalhar com o Fabric de maneira local:
* **Hyperledger Fabric** -> BC completa (Setup mais complexo mas é prod ready)
* **Microfab** -> Container docker rodando fabric em memória (setup simples), não é indicado para produção
* **MiniFab** -> Script + Docker - Fabric real porém com setup automatizado (também não é 100% indicado pra prod)
